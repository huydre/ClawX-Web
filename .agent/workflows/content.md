## Workflow

1. If the user provides screenshots or videos, analyze and describe the context in detail.
2. Research for relevant information, trends, competitor analysis.
3. Search the codebase for context if the content relates to the product.
4. Plan the copy — make sure it satisfies the user request.
5. Apply `copywriter` agent expertise to write the copy based on the plan.
6. Present to user for review.

---

## Copy Quality Standards

- **Hook First**: First 5 words determine if they read the next 50
- **Specificity Wins**: "47% increase" beats "boost results"
- **No Corporate Speak**: Write like texting a smart friend
- **No Hashtag Spam**: Sparingly or not at all
- **A/B Variants**: Always provide 2-3 versions to test

## Frameworks
- **AIDA**: Attention → Interest → Desire → Action
- **PAS**: Problem → Agitate → Solution
- **BAB**: Before → After → Bridge
