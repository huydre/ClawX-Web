# Document Skills

Process and extract data from various document formats.

## Supported Formats

### PDF (`pdf/`)
- Extract text, tables, and metadata from PDF files
- OCR support for scanned documents

### DOCX (`docx/`)
- Parse Word documents
- Extract text, images, tables, and formatting

### PPTX (`pptx/`)
- Parse PowerPoint presentations
- Extract slides, text, images, and speaker notes

### XLSX (`xlsx/`)
- Parse Excel spreadsheets
- Extract sheets, cells, formulas, and charts

## When to Use
- Reading and parsing document uploads
- Extracting structured data from files
- Document format conversion
- Data migration from documents to structured formats
