# Chat AI

Giao diện Chat là nơi bạn trò chuyện trực tiếp với AI — giống như nhắn tin trên Messenger hay Zalo.

## Cách chat cơ bản

1. Nhấn vào mục **Chat** trên thanh bên (hoặc thanh dưới trên điện thoại)
2. Nhập tin nhắn vào ô ở cuối màn hình
3. Nhấn **Enter** hoặc nút gửi
4. Chờ AI trả lời — tin nhắn sẽ xuất hiện dần dần theo thời gian thực

::: tip Mẹo nhập tin nhắn
- Nhấn **Enter** để gửi tin nhắn
- Nhấn **Shift + Enter** để xuống dòng mà không gửi
:::

## Gửi file và hình ảnh

Bạn có thể gửi file cho AI xử lý bằng nhiều cách:

- **Nút đính kèm** — Nhấn biểu tượng 📎 cạnh ô nhập liệu để chọn file
- **Kéo và thả** — Kéo file từ máy tính vào cửa sổ chat
- **Dán từ clipboard** — Copy hình ảnh rồi nhấn Ctrl+V (hoặc Cmd+V trên Mac)

Các loại file được hỗ trợ:
- Hình ảnh (PNG, JPG, WebP...)
- Video, âm thanh
- Tài liệu (PDF, Word, Excel...)
- File nén (ZIP, RAR...)
- File văn bản, code

## Quản lý phiên chat

### Tạo phiên chat mới
Nhấn nút **"+"** hoặc **"Phiên mới"** trên thanh công cụ để bắt đầu cuộc trò chuyện mới. Phiên cũ vẫn được lưu.

### Chuyển đổi giữa các phiên
Nhấn vào menu dropdown phiên chat (góc trên) để xem và chuyển sang phiên khác. Mỗi phiên có ngữ cảnh riêng biệt — AI sẽ nhớ những gì bạn đã nói trong phiên đó.

### Làm mới
Nhấn nút làm mới (🔄) để tải lại lịch sử tin nhắn nếu có vấn đề hiển thị.

## Xem quá trình suy luận (Thinking)

Nhấn biểu tượng **bộ não** (🧠) trên thanh công cụ để bật/tắt hiển thị quá trình suy luận. Khi bật, bạn sẽ thấy AI "nghĩ gì" trước khi trả lời — phần này có thể thu gọn/mở rộng.

## Xem AI sử dụng công cụ

Khi AI cần thực hiện tác vụ (ví dụ: tìm kiếm web, thực thi lệnh, đọc file), bạn sẽ thấy thẻ hiển thị:
- Tên công cụ đang dùng
- Dữ liệu đầu vào
- Kết quả trả về

Điều này giúp bạn hiểu AI đang làm gì.

## Dừng AI giữa chừng

Nếu AI đang trả lời nhưng bạn muốn dừng, nhấn nút **Dừng** (⏹️) ở cuối màn hình. AI sẽ ngừng xử lý ngay lập tức.

## Hiển thị tin nhắn

Tin nhắn của AI hỗ trợ định dạng phong phú:
- **In đậm**, *in nghiêng*, `code`
- Danh sách, bảng biểu
- Khối code với tô màu cú pháp
- Hình ảnh (nhấn vào để xem lớn)
- Liên kết có thể nhấn

Mỗi tin nhắn AI đều có nút **Sao chép** để copy nội dung.

## Xem model AI đang dùng

Trên thanh công cụ, bạn sẽ thấy tên model AI hiện tại (ví dụ: "GPT-4" hoặc "Claude"). Để đổi model, vào **Cài đặt > Nhà cung cấp AI**.

## Xử lý lỗi

Nếu có lỗi xảy ra (ví dụ: mất kết nối, API key hết hạn), một thanh lỗi sẽ hiện ra với:
- Mô tả lỗi
- Nút **Thử lại** — gửi lại tin nhắn
- Nút **Bỏ qua** — đóng thông báo lỗi
